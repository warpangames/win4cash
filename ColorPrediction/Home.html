How to Play

1 minute 1 issue, 50 seconds to order, 10 seconds waiting for the draw. It opens all day.

Color Bet Rules:

  • Select Green: If the result shows green you will get return multiple of your bet amount.
  • Select Red: If the result shows red you will get return multiple of your bet amount.
  • Select blue: If the result shows blue you will get return multiple of your bet amount.

Number Bet Rules:

Select a specific number (0-9). If the result shows your selected number, you will get return multiple of your bet amount.

Big/Small Bet Rules:

  • Big: If the result shows numbers from 5 to 9, return multiple of your bet amount.
  • Small: If the result shows numbers from 0 to 4, return multiple of your bet amount.
